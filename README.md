# Avian Phylogeny Project
This is a project for CS 466 (Introduction to Bioinformatics) where we seek to illustrate the divergence between paleognaths and neognaths via the UPGMA hierarchical clustering algorithm applied to RAG1 gene sequences.

[![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/d-tom75/avian-phylogeny-project/blob/main/cs466_project_analysis.ipynb)

## Running the code
No installation is necessary. Simply click the **Open in Colab** badge directly above, and run all the cells in order. The notebook will automatically clone the repository and load the sequence data from 'data/', which will be used to generate a phylogeny.
